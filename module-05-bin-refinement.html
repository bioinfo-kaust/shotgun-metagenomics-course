MODULE 5 — Day 3, Late Morning

Bin Refinement

Combine and de-replicate bins from multiple binners using DAS Tool to produce a single optimised, non-redundant set of high-quality MAGs.

Learning Objectives

  • Understand why combining multiple binners improves MAG quality and recovery
  • Prepare scaffold-to-bin mapping files for DAS Tool from each binner
  • Run DAS Tool to select the optimal non-redundant bin set
  • Interpret DAS Tool scoring metrics

Background

Different binning algorithms capture complementary signals and each performs best on different community structures. Running multiple binners and combining their outputs with DAS Tool consistently recovers more and higher-quality MAGs than any single binner. DAS Tool uses a scoring function based on single-copy gene completeness and contamination estimates to select the best non-overlapping set of bins from across all binner outputs. It does not create new bins — it selects the best existing ones.

Hands-on

Step 1 — Generate Scaffold-to-Bin Files

DAS Tool requires a two-column TSV (scaffold → bin) for each binner:

mkdir -p results/05-refinement/scaffold_lists

# MetaBAT2
Fasta_to_Scaffolds2Bin.sh \
    -i results/04-binning/metabat2/ \
    -e fa \
    > results/05-refinement/scaffold_lists/metabat2.tsv

# MaxBin2
Fasta_to_Scaffolds2Bin.sh \
    -i results/04-binning/maxbin2/ \
    -e fasta \
    > results/05-refinement/scaffold_lists/maxbin2.tsv

# CONCOCT
Fasta_to_Scaffolds2Bin.sh \
    -i results/04-binning/concoct/bins/ \
    -e fa \
    > results/05-refinement/scaffold_lists/concoct.tsv

# SemiBin2
Fasta_to_Scaffolds2Bin.sh \
    -i results/04-binning/semibin2/output_bins/ \
    -e fa \
    > results/05-refinement/scaffold_lists/semibin2.tsv

Step 2 — Run DAS Tool

mkdir -p results/05-refinement/das_tool

DAS_Tool \
    -i results/05-refinement/scaffold_lists/metabat2.tsv,\
results/05-refinement/scaffold_lists/maxbin2.tsv,\
results/05-refinement/scaffold_lists/concoct.tsv,\
results/05-refinement/scaffold_lists/semibin2.tsv \
    -l metabat2,maxbin2,concoct,semibin2 \
    -c results/02-assembly/megahit/sample.contigs.fa \
    -o results/05-refinement/das_tool/sample \
    --write_bins \
    --write_unbinned \
    --score_threshold 0.5 \
    --threads 8
Score threshold The default --score_threshold 0.5 retains bins where completeness minus 5× contamination exceeds 0.5. Lowering this recovers more bins at lower quality; raising it (e.g. 0.7) produces fewer but higher-confidence bins.

Step 3 — Review DAS Tool Scores

# View summary of selected bins
cat results/05-refinement/das_tool/sample_DASTool_summary.tsv

# Count selected bins
tail -n +2 results/05-refinement/das_tool/sample_DASTool_summary.tsv | wc -l

# List bins by score (higher = better)
sort -t$'\t' -k11 -rn \
    results/05-refinement/das_tool/sample_DASTool_summary.tsv | \
    head -20

Step 4 — Inspect Refined Bins

# List all selected bins
ls results/05-refinement/das_tool/sample_DASTool_bins/

# Check sizes
for f in results/05-refinement/das_tool/sample_DASTool_bins/*.fa; do
    echo -e "$(basename $f)\t$(grep -c '^>' $f) contigs\t$(awk '/^>/{next}{sum+=length($0)}END{print sum}' $f) bp"
done
