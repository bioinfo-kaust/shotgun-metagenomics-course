What You Will Learn

Day 1 labs cover the cluster environment and pipeline overview. Modules 1–8 each cover one stage of the nf-core/mag pipeline — you run the tools manually first, then see how the workflow ties it all together.

DAY 1 LAB

Linux Command Line & Ibex Cluster

Connect to the Ibex HPC cluster, navigate the filesystem, use essential Linux commands for file inspection, activate conda environments, and submit and monitor SLURM jobs.

SSHLinuxcondaSLURMscreen
DAY 1 LAB

Nextflow & nf-core/mag Overview

Understand the nf-core/mag pipeline structure, run a stub dry-run to trace the workflow graph, and submit a full pipeline job on Ibex using a SLURM batch script.

Prerequisites

This is an intermediate-level course. Participants should have the following background.

Linux Command Line

  • Navigate the filesystem
  • Run and chain commands
  • Manage conda environments

NGS Concepts

  • Short-read sequencing basics
  • FASTQ format and quality scores
  • Paired-end reads

Compute Resources

  • ≥16 GB RAM (≥32 GB for assembly)
  • ≥8 CPU cores recommended
  • ~200 GB free disk space
