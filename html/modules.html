Course Modules

8 hands-on modules following the nf-core/mag workflow stage by stage, covering short reads, long reads, and ancient DNA.

DAY 1 LAB

Linux Command Line & Ibex Cluster

Connect to the Ibex HPC cluster, navigate the filesystem, use essential Linux commands for file inspection, activate conda environments, and submit and monitor SLURM jobs.

SSHLinuxcondaSLURMscreen
DAY 1 LAB

Nextflow & nf-core/mag Overview

Understand the nf-core/mag pipeline structure, run a stub dry-run to trace the workflow graph, and submit a full pipeline job on Ibex using a SLURM batch script.

Nextflownf-core/magSLURM
MODULE 1 — Day 1

QC & Preprocessing

Quality-assess, trim adapters, remove host and PhiX contamination, and normalise coverage for short reads; run adapter removal, quality filtering, and host decontamination for long reads.

FastQCfastpBowtie2BBNormNanoPlotPorechop_ABIChopperFiltlongNanoq
MODULE 2 — Day 2

Assembly, Evaluation & Annotation

Assemble short reads with MEGAHIT and metaSPAdes, long reads with Flye and metaMDBG, or hybrid assemblies. Evaluate with QUAST and ALE, map reads for coverage, predict genes with Prodigal, and identify viral/plasmid sequences with geNomad.

MEGAHITmetaSPAdesFlyemetaMDBGQUASTALEBowtie2Minimap2ProdigalgeNomad
MODULE 3 — Day 2

Ancient DNA Analysis

Identify DNA damage patterns characteristic of ancient samples with PyDamage, call high-confidence variants with FreeBayes, and generate corrected consensus sequences with BCFtools.

PyDamageFreeBayesBCFtools
MODULE 4 — Day 3

Contig Binning

Map reads to assemble coverage profiles, then bin contigs into draft genomes with six complementary binners. Apply Tiara to classify contigs by domain before binning.

MetaBAT2MaxBin2CONCOCTCOMEBinMetaBinnerSemiBin2Tiara
MODULE 5 — Day 3

Bin Refinement

Combine and de-replicate bins from multiple binners using DAS Tool to produce a single optimised, non-redundant set of high-quality MAGs.

DAS Tool
MODULE 6 — Day 3

Bin Quality Evaluation

Measure MAG completeness and contamination with CheckM2 or CheckM, evaluate gene-set completeness with BUSCO, detect chimeric bins with GUNC, and assess assembly quality with QUAST. Apply MIMAG standards.

CheckM2CheckMBUSCOGUNCQUAST
MODULE 7 — Day 4

Taxonomy & Annotation

Classify MAGs phylogenomically using the GTDB reference tree with GTDB-Tk, annotate contigs and bins with CAT, predict prokaryotic genes with Prokka, and eukaryotic genes with MetaEuk.

GTDB-TkCATProkkaMetaEuk
MODULE 8 — Day 4

Run Summary

Aggregate QC and tool metrics with MultiQC, generate per-sample MAG recovery tables with Bin Summary, and explore the full MAG catalogue with BigMAG. Run the complete nf-core/mag pipeline end-to-end.

MultiQCBin SummaryBigMAGnf-core/mag
