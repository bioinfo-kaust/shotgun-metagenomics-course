Resources

Tools, key papers, databases, and links used throughout the course.

Tools by Module

Every tool used in this course, organised by the pipeline stage where it appears.

Module 1 — QC & Preprocessing

  • FastQC — Raw read QC reports
  • fastp — Adapter trimming & filtering (default)
  • AdapterRemoval2 — Alternative trimmer for aDNA
  • Trimmomatic — Alternative Illumina trimmer
  • BBnorm — Digital normalisation
  • MultiQC — Aggregate QC reports

Module 2 — Host Read Removal

  • Bowtie2 — Short-read aligner (host/PhiX)
  • SAMtools — BAM/SAM processing
  • NanoPlot — Long-read QC plots
  • NanoLyse — Lambda phage removal (long reads)
  • Porechop_abi — Nanopore adapter trimming
  • Chopper — ONT quality filtering
  • Nanoq — PacBio quality filtering
  • Filtlong — Long-read length filtering

Module 3 — Assembly

  • MEGAHIT — Fast short-read assembler
  • metaSPAdes — High-quality short-read assembler
  • SPAdes Hybrid — Short + long read assembler
  • Flye — Long-read assembler
  • metaMDBG — PacBio HiFi / ONT r10 assembler
  • QUAST — Assembly quality assessment
  • ALE — Assembly likelihood evaluation

Module 4 — Gene Prediction & Annotation

  • Prodigal — Prokaryotic gene prediction
  • Prokka — Rapid genome annotation
  • MetaEuk — Eukaryotic gene prediction
  • geNomad — Virus & plasmid identification

Module 5 — Binning

  • MetaBAT2 — Coverage + composition (default)
  • MaxBin2 — EM algorithm binner
  • CONCOCT — Coverage + k-mer PCA
  • COMEBin — Contrastive multi-view learning
  • MetaBinner — Ensemble binning
  • SemiBin2 — Self-supervised deep learning
  • DAS Tool — Bin refinement & dereplication

Module 6 — Bin Quality Assessment

  • CheckM2 — ML-based completeness & contamination
  • CheckM — Marker-gene MAG quality
  • BUSCO — Gene set completeness
  • GUNC — Chimerism & contamination screening

Module 7 — Taxonomy Assignment

  • GTDB-Tk — Phylogenomic taxonomy (default)
  • CAT / BAT — Contig & bin protein-level taxonomy
  • Tiara — Deep-learning domain classification

Module 8 — nf-core/mag & Special Modes

  • Nextflow — Workflow engine
  • nf-core/mag v5.4.2 — Full automated pipeline
  • PyDamage — Ancient DNA damage authentication
  • FreeBayes — aDNA damage correction (variant calling)
  • BIgMAG — MAG summary dashboard
